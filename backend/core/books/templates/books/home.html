Welcome to BookDB

A simple Django REST API demonstrating CRUD operations with nested relationships.

CRUD Operations

  • CREATE: Add new authors and books via /admin/ or REST API
  • READ: View data on this site, API, or admin interface
  • UPDATE: Edit authors and books via /admin/ or REST API
  • DELETE: Remove data via /admin/ or REST API

Access Points

  • Web Views: Browse authors and books (read-only display)
  • REST API: Full CRUD at /home/api/authors/ and /home/api/books/
  • Admin Panel: Full CRUD with UI at /admin/
  • API Docs: Interactive Swagger UI at /swagger/
